# Shared Types Package

This package contains TypeScript types shared between the frontend and backend.

## Types

- `BlockState` - Represents a single effect block
- `DeviceState` - Represents the entire device state
- `ServerMsg` - WebSocket messages from server to client
- `ClientMsg` - WebSocket messages from client to server
- `PresetFile` - Preset file format for disk storage

## Building

```bash
npm run build
```

## Watching for changes

```bash
npm run watch
```
